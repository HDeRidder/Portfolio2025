Project IOT

Course: Pumpkin Regatta at Kasterlee

Assignment: Automated a pumpkin greenhouse for the pumpkin regatta at Kasterlee.

Technology:

  • #ESP32
  • #SensorIntegration
  • #RaspberryPi
  • #HomeAutomation
  • #Python

Team: Harold De Ridder, Robbe Van Hout, Robbe Vangeel, Dylan Van den Broeck

What did I apply?

  • Deployed soil moisture, temperature, and humidity sensors throughout the greenhouse
  • Automated irrigation system based on real-time sensor data
  • Remote monitoring and control via a web dashboard
  • Data logging for growth analysis and optimization
  • Integration with weather forecast APIs for proactive adjustments
  • Fully automated irrigation and climate control
  • Real-time alerts for abnormal conditions
  • Historical data visualization for crop management
  • Mobile-friendly interface for on-the-go monitoring

Context & Background

This project focused on automating a pumpkin greenhouse for the pumpkin regatta at Kasterlee.

Results

The greenhouse automation system led to healthier pumpkin growth, reduced water usage, and less manual intervention. The project was showcased at the pumpkin regatta, demonstrating the benefits of IoT in modern agriculture.
