Project IOT

Course: Pumpkin Regatta at Kasterlee

Assignment: Automated a pumpkin greenhouse for the pumpkin regatta at Kasterlee.

Technology:

  • #ESP32
  • #SensorIntegration
  • #RaspberryPi
  • #HomeAutomation
  • #Python

Team: Harold De Ridder, Jens Geenen, Jens Ceusters, Lucas De Greef, Mathieu Du Jardin

Context & Background

This project focused on automating a pumpkin greenhouse for the pumpkin regatta at Kasterlee. The main objective was to create an intelligent system that could monitor and control environmental factors within the greenhouse, ensuring optimal conditions for pumpkin growth with minimal human intervention.

Realization

We deployed soil moisture, temperature, and humidity sensors throughout the greenhouse to collect real-time data. An automated irrigation system was developed to deliver water based on this sensor data, ensuring efficient water usage. We also implemented a remote monitoring and control system accessible via a web dashboard, allowing for adjustments from anywhere. Data logging capabilities were included for growth analysis and optimization, and the system was integrated with weather forecast APIs for proactive adjustments to climate control. This resulted in a fully automated irrigation and climate control system with real-time alerts for abnormal conditions.

What did I apply?

  • Deployed and integrated various environmental sensors.
  • Developed an automated irrigation system based on real-time data.
  • Created a web dashboard for remote monitoring and control.
  • Implemented data logging for growth analysis and optimization.
  • Integrated weather forecast APIs for proactive climate adjustments.
  • Developed a mobile-friendly interface for on-the-go monitoring.
