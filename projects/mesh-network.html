Mesh Network

Course: Embedded Devices Essentials

Assignment: Created a mesh network using ESP32 microcontrollers across the whole campus to measure the temperature of the Hive

Technology:

  • #ESP32
  • #C++
  • #SensorIntegration

Team: Harold De Ridder, Jens Geenen, Mathieu Dujardin

What did I apply?

  • ESP32 microcontrollers configured as mesh nodes
  • Custom firmware for dynamic routing and self-healing
  • Real-time monitoring dashboard
  • Secure communication protocols
  • Scalable architecture for future expansion
  • Automatic node discovery and integration
  • Fault tolerance and self-healing capabilities
  • Low-latency communication between nodes
  • Easy deployment and management

Context & Background

This project involved building a robust mesh network across campus using ESP32 microcontrollers. The goal was to create a resilient, self-healing wireless network that could support IoT devices and enable reliable communication even in challenging environments.

Results

The mesh network successfully demonstrated reliable communication across campus, with seamless integration of new nodes and robust performance even in the event of node failures. The project highlighted the potential of mesh networking for scalable IoT deployments.
